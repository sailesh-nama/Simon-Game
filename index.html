Simon Game

Rules

  • This game tests your memory skill, be ready to remember the sequence.
  • Simon gives you a color signal as soon as you start the game, and you should repeat the signal by clicking the same color.
  • Now in second level it again gives you a new color signal, this time you should press both the first and second color.
  • So in each level it gives you a new color signal and you should repeat all the color sequence from first along with the new color.
  • Lets see how long you can remember the color pattern.
